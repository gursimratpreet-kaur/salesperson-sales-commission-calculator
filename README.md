# salesperson-sales-commission-calculator
Java code to enter information about salesperson and calculate commissions 
The program begins by asking for salesperson's ID or -1 to end
Then it asks for name and district which is analysed using an enum.
After this it asks for sales for various fruits and then asks again for ID.
If new ID is entered it begins the same process again if -1 is entered it calculates commission and outputs in the form of table.
